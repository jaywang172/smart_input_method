### 階段 2：安裝測試

#### 2.1 安裝到系統
```bash
# 方法 1：從 DMG 拖曳
open build/SmartInputMethod-1.0.0.dmg
# 然後手動拖曳 SmartInputMethod.app 到 Applications

# 方法 2：使用命令行
cp -R build/SmartInputMethod.app /Applications/
```

**預期結果**：
- ✅ App 成功複製到 /Applications
- ✅ 在 Launchpad 中可見
- ✅ 可以從 Finder 啟動

---

#### 2.2 啟用輸入法

1. 打開「系統偏好設定」→「鍵盤」→「輸入法」
2. 點擊左下角「+」按鈕
3. 找到「智能輸入法」
4. 點擊「加入」

**預期結果**：
- ✅ 輸入法出現在列表中
- ✅ 可以切換到智能輸入法
- ✅ 選單列顯示輸入法圖標

### 階段 3：功能測試

#### 3.1 基本輸入測試

打開任何文字編輯器（如 TextEdit），切換到智能輸入法：

**測試案例 1：注音輸入**
```
輸入: su3cl3
預期: 你好
```

**測試案例 2：英文輸入**
```
輸入: hello
預期: hello
```

**測試案例 3：混合輸入**
```
輸入: hello su3cl3
預期: hello 你好
```

**測試案例 4：長句子**
```
輸入: rup wu0 wu0 fu45p cl3
預期: 今天天氣真好
```
