## 🏆 核心演算法

### 1. Trie 樹 (前綴樹)
```
時間複雜度: O(m) 查找
空間複雜度: O(N×M×C)
用途: 快速詞彙查找、自動補全
```

### 2. N-gram 語言模型
```
類型: Unigram, Bigram, Trigram
平滑: Laplace Smoothing
用途: 上下文預測、候選詞排序
```

### 3. Viterbi 演算法
```
時間複雜度: O(n×s²)
空間複雜度: O(n×s)
用途: 找出最佳注音轉換路徑
```

### 4. 動態規劃
```
應用: 注音分段、路徑優化
優化: 剪枝、Top-K 保留
```

### 5. 樸素貝葉斯分類器
```
類型: 多項式樸素貝葉斯
特徵: 8+ 維特徵向量
用途: 語言分類
```
